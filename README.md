# 疫情防控管理系统 java940

## 项目概述

疫情防控管理系统是一款基于Spring Boot开发的系统，旨在为用户提供便捷的疫情管理功能。系统主要分为用户和管理员两个角色，分别拥有不同的功能模块。

## 技术栈

- **后端：**
  - Spring Boot
  - Spring
  - Spring MVC
  - MySQL
  - MyBatis
- **前端：**
  - HTML
  - CSS

## 运行环境

- Java 1.8+
- MySQL 5.7+
- Maven 3

## 功能模块

### 管理员

- 管理登录用户

### 用户

- 医院管理
- 密切接触管理
- 确诊管理
- 治愈管理
- 遇难管理

## 系统角色

- 管理员：负责管理登录用户及系统维护
- 用户：负责疫情相关信息的管理

## 核心亮点

- 使用Spring Boot框架，简化系统开发流程
- 前后端分离，易于维护和扩展
- 针对不同角色设计相应功能模块，高效管理疫情信息
- 基于MySQL和MyBatis，保证数据存储和访问的稳定性和高效性
- 适用于Java 1.8以上版本，具备良好的跨平台性能
- 使用Maven构建项目，便于依赖管理和自动化构建
